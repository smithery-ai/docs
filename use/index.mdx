Smithery provides AI agent developers with access to a large ecosystem of Model Context Protocol (MCP) servers. Whether you're building chatbots, background agents, or AI copilots, Smithery simplifies MCP integration through our unified gateway.

**Benefits of using Smithery:**

- **Server Registry**: Access thousands of MCP servers built by the community
- **Unified Gateway**: Connect to MCP servers through a single, unified API
- **Managed Authentication**: Let Smithery handle OAuth flows and API key management
- **Configuration Profiles**: Save and reuse server configurations across projects
