## Introduction

ChatGPT apps are interactive widgets that run inside AI conversations, built with React and powered by the Model Context Protocol (MCP). Instead of text-only responses, your MCP server can display rich, interactive user interfaces that respond to user actions in real-time.

<img src="/images/apps.png" alt="ChatGPT apps example" style={{ border: '1px solid #e5e7eb', borderRadius: '8px' }} />

### What You Can Build

With ChatGPT apps, you can create:

- **Interactive dashboards** - Kanban boards, task managers, project trackers
- **Data visualizations** - Charts, graphs, maps, and custom data displays
- **Forms and data entry** - User input interfaces with validation
- **Games and experiences** - Interactive games, quizzes, and educational tools
- **Collaboration tools** - Real-time interfaces for team workflows
