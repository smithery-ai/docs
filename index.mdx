## What can I do with Smithery?

You can use Smithery to build, find, and use MCP servers. We host popular MCP servers like:

- [Exa](https://smithery.ai/server/exa) - Search the live web, access LinkedIn profiles, do deep research, and more
- [Context7](https://smithery.ai/server/@upstash/context7-mcp) - Reference the latest docs for most major SDKs and frameworks directly in Cursor or Claude Code
- [Browserbase](https://smithery.ai/server/@browserbasehq/mcp-browserbase) - Control a remote web browser using [Stagehand](https://stagehand.dev)
