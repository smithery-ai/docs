## What can I do with Smithery?

You can use Smithery to build, find, and use MCP servers. We host popular MCP servers like:

- [Exa](https://smithery.ai/server/exa) - Search the live web, access LinkedIn profiles, do deep research, and more
- [Context7](https://smithery.ai/server/@upstash/context7-mcp) - Reference the latest docs for most major SDKs and frameworks directly in Cursor or Claude Code
- [Browserbase](https://smithery.ai/server/@browserbasehq/mcp-browserbase) - Control a remote web browser using [Stagehand](https://stagehand.dev)

## LLM-Friendly Docs

The Smithery documentation is available in multiple LLM-friendly formats:

### MCP Server

The Smithery docs are accessible via MCP at `https://smithery.ai/docs/mcp`. You can also use the [Context7 MCP server](https://smithery.ai/server/@upstash/context7-mcp) to search the Smithery documentation.

<Tip>
  Install the [Context7 MCP
  server](https://smithery.ai/server/@upstash/context7-mcp) and use this prompt:
  ``` /context7 how do I build an MCP server using smithery? ``` ![Context7
  Smithery Integration](/images/context7-smithery-2.png)
</Tip>
