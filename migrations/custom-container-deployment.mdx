## Overview

Custom container deployment is ideal when you:
- Use languages other than TypeScript (Python, Go, Rust, etc.)
- Need full control over your Docker environment
- Have complex dependencies or custom build processes
- Want to implement HTTP transport manually

## Key Requirements

Your custom container must:
1. **HTTP Endpoint**: Serve `/mcp` endpoint handling `GET`, `POST`, `DELETE` requests
2. **Port Configuration**: Listen on `PORT` environment variable
3. **Configuration Parsing**: Parse base64 encoded config from query parameters
4. **MCP Protocol**: Implement [Streamable HTTP](https://modelcontextprotocol.io/specification/2025-03-26/basic/transports#streamable-http) transport
